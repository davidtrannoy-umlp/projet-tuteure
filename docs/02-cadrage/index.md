# Objectifs et livrables

## Objectif général du projet

Refondre le site [infocombesancon.fr](https://infocombesancon.fr) avec **Elementor**, avec une forte attention sur le responsive et la réorganisation des contenus, sur la base d'une **analyse préalable** des cibles et des objectifs du site.

## Phases du projet

Le projet se déroule en grandes phases, à affiner en équipe dès le démarrage :

1. **Cadrage** — comprendre le contexte, les cibles, les objectifs (ce document en est le point de départ)
2. **Analyse préalable** — audit du site existant, recherche de cibles, benchmark (voir [Analyse préalable](../03-analyse-prealable/index.md))
3. **Conception** — arborescence cible, wireframes, choix éditoriaux
4. **Réalisation** — intégration sous Elementor (voir [Refonte](../04-refonte/index.md))
5. **Tests et recette** — vérification responsive, relecture des contenus, validation
6. **Livraison**

!!! warning "Dans un premier temps : cadrer les besoins"
    La priorité du tout début de projet n'est **pas** de commencer à designer dans Elementor, mais de **cadrer les besoins** : qui visite le site, pourquoi, quels sont les objectifs prioritaires. Voir la page [Analyse préalable](../03-analyse-prealable/index.md).

## Livrables attendus

Pour le 10 novembre, les livrables attendus sont :

- Un dossier d'analyse préalable (cibles, objectifs, audit, benchmark)
- Une arborescence cible du nouveau site
- Les maquettes des pages clés

Pour le 30 mars (date à valider) : 

- Le site refondu sous Elementor, fonctionnel et responsive
- Une documentation de passation pour les futurs administrateurs du site

Détails techniques de ces livrables : voir [Livrables attendus](../04-refonte/livrables-attendus.md).
