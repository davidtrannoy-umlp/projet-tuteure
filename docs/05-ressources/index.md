## Outils suggérés

- **Elementor** — page builder WordPress utilisé pour la refonte
- **Figma** ou équivalent — pour les maquettes (optionnel selon préférence de l'équipe)
- **Google Sheets / Excel** — pour l'inventaire de contenu et l'audit
- **PageSpeed Insights** — pour tester la performance du site
- Outils de test responsive : DevTools du navigateur (mode responsive), ou test sur appareils réels

## Glossaire rapide

| Terme                      | Définition                                                                                                |
| -------------------------- | --------------------------------------------------------------------------------------------------------- |
| **Builder / Page builder** | Outil permettant de construire visuellement des pages web sans coder (ex : Elementor)                     |
| **Responsive**             | Capacité d'un site à s'adapter correctement à toutes les tailles d'écran (mobile, tablette, desktop)      |
| **Persona**                | Profil type représentant un groupe d'utilisateurs cibles, utilisé pour guider les décisions de conception |
| **Arborescence**           | Structure hiérarchique des pages d'un site (plan de navigation)                                           |
| **Wireframe**              | Maquette simplifiée d'une page, centrée sur la structure plutôt que sur le design final                   |
| **Recette**                | Phase de vérification finale avant mise en ligne (tester que tout fonctionne comme prévu)                 |
