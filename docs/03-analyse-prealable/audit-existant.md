# Audit du site existant

!!! note "Trame à remplir par l'équipe"
    Objectif : dresser un état des lieux objectif du site [infocombesancon.fr](https://infocombesancon.fr) avant toute décision de refonte.

## 1. Inventaire de contenu

Lister toutes les pages du site actuel. Un tableur (Excel/Google Sheets) est souvent plus pratique pour ce travail ; on peut en résumer l'essentiel ici.

| Page / URL | Objectif supposé | Constat | Proposition |
|---|---|---|---|
| *(ex : /admissions)* | *(ex : informer sur les modalités d'inscription)* | *(ex : contenu à jour mais peu visible depuis l'accueil)* | *(garder / fusionner / réécrire / supprimer)* |

!!! tip "Comment faire cet inventaire"
    Parcourir le site page par page (y compris via le plan du site / sitemap s'il existe), et noter systématiquement : l'URL, le titre, la dernière mise à jour visible, et un jugement rapide sur sa pertinence.

## 2. Audit technique

- [ ] Builder / thème WordPress utilisé (à identifier précisément)
- [ ] Version de WordPress et plugins actifs
- [ ] Hébergeur et accès disponibles
- [ ] Présence d'un certificat SSL (https)
- [ ] Vitesse de chargement (ex : test via PageSpeed Insights)
