# Benchmark

!!! note "Trame à remplir par l'équipe"
    Objectif : observer comment d'autres sites comparables traitent les mêmes enjeux (cibles similaires, contenus similaires), pour s'en inspirer — sans copier.

## 1. Sélection des sites à étudier

Choisir 3 à 5 sites pertinents, par exemple :

- D'autres départements ou IUT proposant un BUT Information-Communication
- D'autres formations de l'enseignement supérieur avec un public similaire (lycéens, étudiants)
- Des sites non concurrents mais reconnus pour leur UX (à titre d'inspiration sur un point précis : navigation, responsive, etc.)

## 3. Synthèse

!!! tip "L'objectif n'est pas d'imiter"
    Le benchmark sert à identifier des **patterns qui fonctionnent** (ex : une navigation à 5 entrées maximum, un bouton « Candidater » visible dès l'accueil) — pas à copier un site en particulier. Croisez ces observations avec les besoins identifiés dans [Cibles et objectifs](cibles-objectifs.md).
